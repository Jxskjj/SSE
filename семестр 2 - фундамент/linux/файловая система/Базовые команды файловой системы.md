вВ базовые команды входят команды перемещения по директориям, вывода информации о директории, создание/удаление/копирование/перемещения файлов, а также вывод их информации
Рассмотрим каждую из команд подробно

## 1) CD - change directory
Используется для перемещения по файловой системе. Синтаксис команды: `cd [path]`. Например:
```bash
cd .config/neovim
cd Downloads
cd #перемещение в корневую  директорию пользователя
cd ~/Desktop/
cd . #количество точек влияет на то, насколько глубоко мы спукаемся. одна точка - текущая директория, две точки - на одну директориб вверх и тд
```
## 2) LS - list 
Используется для вывода информации о текущей директории. Синтаксис команды: `ls [опции] [путь]`. 
Приведем пример вывода простого `ls`, а затем разберем опции, и как их можно комбинировать. 
```bash
╭─ jxskj@thinkpad ~
╰─❯ ls
Desktop  Documents  Downloads  Music  Pictures  Public  Templates  Videos
#мы получаем вывод всех директорий и файлов, которые содержаться в нашей директории 
```
Теперь разберем основные  опции и на что они влияют:
1) -l : предоставляет подробную информацию о каждой директории/файле.  Рассмотрим её вывод:
```bash
╭─ jxskj@thinkpad ~
╰─❯ ls -l
total 32
drwxr-xr-x 2 jxskj jxskj 4096 Jan 16 22:33 Desktop
drwxr-xr-x 5 jxskj jxskj 4096 Jan 25 20:48 Documents
drwxr-xr-x 3 jxskj jxskj 4096 Jan 25 20:34 Downloads
drwxr-xr-x 2 jxskj jxskj 4096 Jan  7 23:32 Music
drwxr-xr-x 3 jxskj jxskj 4096 Jan 13 04:12 Pictures
drwxr-xr-x 2 jxskj jxskj 4096 Jan  7 23:32 Public
drwxr-xr-x 2 jxskj jxskj 4096 Jan  7 23:32 Templates
drwxr-xr-x 2 jxskj jxskj 4096 Jan  7 23:32 Videos
#права доступа, владелец, группа, размер, дата и время последного изменение и само название
```
2) -a : выводит информацию включая скрытые файлы/директории. Они начинаются с точки в названии своего имени.
3) -R : рекурсивный вывод. Отображает содержимое каталогов и их подкаталогов. 
4) -t : сортировка по времени
5) -S : сортировка по размеру 
6) -r : инверсия прошлой опции. Например -ltr выведет сначала директории, которые были изменены раньше всех. Также работает и для других опций.

## 3) Работа с файлами и директориями
Сейчас разберем подробно работу команд, отвечающих за создание/удаление/перемещение/копирование файлов и директорий. Разберем сразу в коде для наибольшего понимания.
```bash
touch main.py
touch Downloads/main.py #touch отвечает за создание файлов

mkdir [path] #простое создание директории
mkdir -p ~/Python/prog1 #создание дочерних директорий
mkdir --mode=MODE [path] #MODE задает права доступа, синтаксис такой же как и в chmod. он будет рассмотрен далее

rm -i [file path или директоия] #удаление с запросом на удаление
rm -f [file path или директория] #игнорирование несуществующих файлов
rm -r [file path или директория] #удаление директории и ее содержимого если там что то есть
rm -d [file path или директория] #удаление пустых директорий

cp main.py ~/Python
cp -r ~/progs ~/Python #копирует директорию и всё ее содержимое
cp -i  main.py ~/Python/prog.py #копирование с запросом на перезапись
cp -p config.ini /etc/ #копирование с сохранением атрибутов. То есть сохраняется дата создания, изменения, пользователя, группы и прочего

mv main.py ~/Python/prog.py #перемещение с переименованием
mv -i  main.py ~/Python/prog.py #перемещение с запросом на перезапись
mv -f  main.py ~/Python/prog.py #перемещение игнорируя запрос на перезапись
mv -b main.py prog.py #переименование файла
mv -n  main.py ~/Python/ #никогда не перезаписывать файл если он есть
```

## 4) Cat, head, tail - вывод файлов
Такие важные команды как `cat`, `head`, `tail` используются для вывода содержимого файлов в консоль. У каждой команды есть свои базовые опции. Также разберем их сразу на практике:
```bash
cat main.py #вывод файла
cat -n main.py #вывод строк файла с нумерацией

head main.py #по умолчанию вывод первых 10 строк
head -n [n] main.py #вывод первых n строк
head -c [n] main.py #вывод первых n байт

tail main.py #по умолчанию вывод последних 10 строк
tail -n [n] main.py #вывод последних n строк
tail -f /var/log/syslog #ожидание терминала и вывод новых строк по мере их появление. Тут например будут выводится логи по мере их появления
tail -n +[n] main.py #вывод последующих строк начиная с n-ой строки
```
