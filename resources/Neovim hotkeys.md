### **Режимы и навигация**
- `ESC` или `Ctrl-[` - Normal mode (основной режим)
- `i` - Insert mode в позиции курсора
- `a` - Insert mode после курсора
- `v` - Visual mode (выделение)
- `V` - Visual line mode
- `Ctrl-v` - Visual block mode
- `h/j/k/l` - Движение (←/↓/↑/→)
- `w` / `b` - Слово вперед/назад
- `0` / `$` - Начало/конец строки
- `gg` / `G` - Начало/конец файла
- `Ctrl-d` / `Ctrl-u` - Прокрутка вниз/вверх
- `Ctrl-f` / `Ctrl-b` - Страница вниз/вверх
- `zz` - Центрировать экран на курсоре
### **Работа с файлами**
- `<leader>ff` - Telescope: поиск файлов
- `<leader>fg` - Telescope: live grep (поиск по тексту)
- `<leader>fb` - Telescope: список буферов
- `<leader>fh` - Telescope: help tags
- `:e файл` - Открыть файл
- `:w` - Сохранить
- `:q` - Выйти
- `:wq` или `ZZ` - Сохранить и выйти
- `:qa` - Выйти из всех окон
- `<leader>s` - Сохранить все файлы
### **Окна и вкладки**
- `<leader>ww` - Переключение между окнами
- `<leader>wd` - Закрыть окно
- `<leader>w-` / `<leader>w|` - Разделить по горизонтали/вертикали
- `Ctrl-w h/j/k/l` - Переход между окнами
- `Ctrl-w =` - Выровнять размеры окон
- `:tabnew` - Новая вкладка
- `gt` / `gT` - Следующая/предыдущая вкладка
